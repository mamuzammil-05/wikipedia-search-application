# 🔍 Wikipedia Search Application

This is a simple web application that allows users to **search Wikipedia articles** and view results instantly.

---

## 🚀 What This Project Does

* You type a keyword (example: *India, AI, Cricket*)
* Press **Enter**
* The app shows:

  * 📄 Article Title
  * 🔗 Link to Wikipedia
  * 📝 Short Description

---

## 🛠️ Technologies Used

* **HTML** → Structure of the page
* **CSS** → Styling and design
* **JavaScript** → Logic and functionality
* **Bootstrap** → For layout and responsiveness
* **Fetch API** → To get data from Wikipedia

---

## ⚙️ How It Works (Step-by-Step)

1. User types something in the search box
2. Presses **Enter key**
3. A loading spinner appears ⏳
4. App sends request to Wikipedia API
5. Results are displayed on the screen
